# ohmkit

电阻工具箱 —— 精确分数运算，零浮点误差。

单文件 C 程序，集成了**串并联等效电阻计算**、**Δ-Y 变换**、**桥式电路求解**、**色环电阻编解码**。

```
$ ohmkit "4+((9||27)+(1||27))||27"
等效电阻 = 10
```

## 功能

### 串并联等效电阻（分数精确）

```bash
# || 优先级高于 +
ohmkit "10 + 20 || 30"            # 10 + (20||30) = 22
ohmkit "4.7 || 10.5"              # 浮点输入自动转分数

# 详细步骤
ohmkit -v "(2||3) + 4"            # 逐行显示约分和化简
```

语法：`R1 + R2` 串联，`R1 || R2` 并联（优先级高于 `+`），括号任意深度嵌套。

### Δ-Y 变换

```
          R1                         Ra
         / \                        / \
        /   \                      /   \
     R3 ----- R2      ←→        Rc --- Rb
      (Δ / 三角形)              (Y / 星形)
```

```bash
ohmkit delta 10 20 30             # Δ → Y
# Ra = 10/3  Rb = 10  Rc = 5

ohmkit wye 1 2 3                  # Y → Δ
# R1 = 11/3  R2 = 11  R3 = 11/2
```

结果以精确分数输出，同时附带小数近似值。

### 桥式电路求解

Wheatstone 电桥有 5 个电阻，无法直接用串并联公式化简。程序取左三角做 Δ→Y 变换，化为纯串并联后求解。

**电阻分布（命令行参数顺序）：**

```
       ┌──R1──C──R3──┐
       │      │      │
    A──┤      R5     ├──B
       │      │      │
       └──R2──D──R4──┘

  R1: A→C（左上）   R3: C→B（右上）
  R2: A→D（左下）   R4: D→B（右下）
  R5: C→D（中间桥）
```

**求解过程：**

1. 取左三角 `(R1, R2, R5)`——三条边形成 A→C→D→A 闭环
2. 做 Δ→Y 变换，三条边缩为星形三臂 Ra、Rb、Rc
3. 变换后电路变为纯串并联：上路径 `Ra→Rc→R3`，下路径 `Ra→Rb→R4`
4. 等效电阻 = `Ra + ((Rc + R3) || (Rb + R4))`

```bash
ohmkit bridge 1 2 3 4 5
# 桥式电路: R1=1 R2=2 R3=3 R4=4 R5=5
#   Δ→Y: Ra=1/4  Rb=5/4  Rc=5/8
#   等效电阻 = 170/71
```

**平衡电桥特例**：当 `R1/R2 = R3/R4` 时，R5 两端等电位、无电流。等效电阻可直接简化为 `(R1+R3) || (R2+R4)`，无需 Δ-Y 变换。程序仍然走完整变换路径，结果一致。

### 色环电阻编解码

```bash
# 色环 → 阻值（支持 4 环、5 环，颜色名中英文均可）
ohmkit color red violet yellow gold      # → 270 kΩ ±5%
ohmkit color 棕 黑 红 金                 # → 1 kΩ ±5%

# 阻值 → 最近标准色环（E24 系列）
ohmkit findcolor 4700                    # → 黄 紫 红 金
```

## 为什么分数运算

浮点运算在嵌套电阻计算中会产生累积误差。例如：

```
4 + ((9 || 27) + (1 || 27)) || 27
```

浮点逐层截断的结果是 8.1697 Ω，而正确答案是 **10 Ω**。误差来自中间步骤的无理数截断在后续运算中被放大。

ohmkit 内部全程使用分数——`long long` 分子和分母，每次运算后用辗转相除法（GCD）约分。等效电阻公式 `a||b = ab/(a+b)` 对分数而言恰好只涉及整数乘法和加法，不会产生任何截断。

## 命令参考

| 命令 | 用法 |
|------|------|
| 表达式 | `ohmkit "10 + 20 \|\| 30"` |
| `-v` | 显示详细化简步骤 |
| `delta` | `ohmkit delta R1 R2 R3`（Δ 三边 → Y 三臂） |
| `wye` | `ohmkit wye Ra Rb Rc`（Y 三臂 → Δ 三边） |
| `bridge` | `ohmkit bridge R1 R2 R3 R4 R5` |
| `color` | `ohmkit color 色1 色2 色3 [色4] [色5]` |
| `findcolor` | `ohmkit findcolor <阻值>` |
